📋 FOR YOUR FRIEND (SPRING BOOT BACKEND)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Share this with your friend who has the Spring Boot backend:

1. ENABLE CORS (Add this to Spring Boot):

@Configuration
public class CorsConfig {
    @Bean
    public WebMvcConfigurer corsConfigurer() {
        return new WebMvcConfigurer() {
            @Override
            public void addCorsMappings(CorsRegistry registry) {
                registry.addMapping("/api/**")
                    .allowedOrigins("http://localhost:3000")
                    .allowedMethods("GET", "POST", "PUT", "DELETE")
                    .allowedHeaders("*");
            }
        };
    }
}

2. RUN SPRING BOOT on: http://localhost:8080

3. PROVIDE API ENDPOINTS:
   - GET /api/buses/running
   - GET /api/buses/track/{busNumber}
   - POST /api/routes/search
   - GET /api/stations
   - POST /api/fare/calculate
   - POST /api/feedback
